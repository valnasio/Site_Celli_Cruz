Tutorial: Como gerenciar seus dados

Este site agora funciona de forma 100% local, utilizando o arquivo imoveis.json como banco de dados.

  • Exportar: Gera um arquivo com todos os seus imóveis, fotos e configurações atuais. Útil para backup.
  • Importar: Permite que você carregue um arquivo JSON anterior. Cuidado: isso substituirá todos os dados atuais!
  • Imagens: Ao fazer upload de uma imagem, ela é salva automaticamente na pasta assets/uploads/ do seu servidor.
