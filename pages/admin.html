Importar / Exportar Dados

Como funciona a persistência O site usa o arquivo data/imoveis.json como banco de dados. Para salvar as alterações feitas neste admin:

Novo - Salvar Automático: Clique em "Salvar Alterações" para exportar e salvar automaticamente

Alternativa - Exportar Manual:
1. Faça as edições nos imóveis
2. Clique em "Baixar imoveis.json"
3. Substitua o arquivo data/imoveis.json pelo arquivo baixado

Para produção, considere configurar um backend PHP/Node para gerenciar dados automaticamente (ver Ajuda).

Salvar no Servidor

Exporta e salva automaticamente no arquivo data/imoveis.json.

Exportar

Baixa o arquivo JSON com todas as alterações feitas nesta sessão.

Importar

Carrega um arquivo JSON para substituir os dados atuais.

Como alterar informações

Imóveis: Vá para "Imóveis" clique no do imóvel edite clique "Salvar" exporte o JSON.

Fotos: Cole a URL da imagem no campo "URL da Imagem". Use serviços como Imgur, Cloudinary, ou suas próprias fotos hospedadas.

Config geral: Vá em "Configurações" → edite telefone, WhatsApp, e-mail, etc.

Preview: Ao editar um imóvel, você vê uma prévia do card em tempo real no modal.

Integração com E-mail

Para receber os formulários por e-mail, siga um destes passos:

Opção 1 - Formspree (gratuito, sem código):
1. Crie conta em formspree.io
2. Crie um formulário e pegue o ID (ex: xabc1234)
3. Em js/main.js, substitua simulateEmailSend por:
fetch('https://formspree.io/f/SEU_ID', { method:'POST', body: JSON.stringify(data), headers:{'Content-Type':'application/json'} })
Opção 2 - EmailJS (gratuito): emailjs.com
Opção 3 - Backend PHP: Ver exemplo no arquivo js/admin.js

Backend para salvar dados automaticamente

Crie o arquivo api/save.php no servidor:

<?php
header('Content-Type: application/json');
header('Access-Control-Allow-Origin: *');
header('Access-Control-Allow-Methods: POST');

$data = file_get_contents('php://input');
json_decode($data); // valida JSON

if (json_last_error() === JSON_ERROR_NONE) {
  file_put_contents('../data/imoveis.json', $data);
  echo json_encode(['ok' => true]);
} else {
  http_response_code(400);
  echo json_encode(['error' => 'JSON inválido']);
}
?>
